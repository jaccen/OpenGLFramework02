フォントを気に入って下さってありがとうございます。
必ず下記をお読みになってからお使い下さい。
使い方の分からない場合、ご使用をお控え下さい。


◆あんずもじはWindows用のTrueTypeフォントです。
　武蔵システム（http://musashi.or.tv/）のソフト、TTEditにて作成しています。


◆「あんずもじ」「あんずもじ等幅」「あんずもじ奏」「あんずもじ湛」は
　フリーフォントですが、著作権は放棄していません。
　再配布・販売・改造は絶対にしないでください。


◆各フォントは、シフトJISに含まれる全ての漢字を書くことができます。


◆当フォント（Windows用のTrueTypeフォント）を
　Mac用やWebフォント用にファイル変換してお使いになることは可能です。
　変換後のフォントを配布・他人に譲渡することはしないでください。


◆ホームページやブログ（個人・商用とも）にてフォントをお使いの際は、
　リンクしていただけると、とても嬉しいです。
　強制ではありませんので、出来ない場合でもフォントはご愛用下さい☆


◆素材サイト（バナー作成・ロゴ作成サイトを含みます）にて、
　配布用素材にフォントをお使いになる場合には、リンクウェアといたします。
　リンク（URL）は　http://www8.plala.or.jp/p_dolce/　にお願いします。
　リンクは同梱のあんずもじのバナーか、テキストでお願いします。


◆印刷物にも（オフラインでの利用全般）ご自由にお使いいただけます。
　商用・非商用を問わずお使い下さい。
　事前または事後のご連絡は不要ですが♪使用報告はとても嬉しいです♪
　書籍・商品でお使いの場合、見本としてひとつ（１冊）いただけるとすっごく嬉しいです☆
　（お知らせだけでも嬉しいです！）


◆スマートフォンのアプリや、ニコニコ動画等の作品にもご自由にお使いいただけます。
　使用の際、「あんずもじ」の表記や等サイトへのリンクはとても嬉しいです。
　リンク出来ない場合でもフォントはご使用いただけますのでご愛用ください☆
　アプリの有料無料は問わず、ご使用可です。


◆Ｗｅｂフォントとしてもご使用いただけます。
　

◆各フォントは絵文字が入っています。「けいせん」の変換で出てきます。
