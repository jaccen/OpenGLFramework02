・ インデントは半角空白4文字分で統一する。
・ ブロック分の中括弧はその分の次の行から用いて、上下をそろえる書き方にする。
   ( 例 ：
 
      void Hoge()
      {
      }
   ）
・ 各ブロックの内容はそのブロックからインデントする。
・ ラベルの後にはインデントする。
・ 長い行は複数行に分割する。
・ 関数やコメントなどは適度に空白を入れ、式の輪郭や引数などを見やすくする。
・ if文やwhile文などでそのブロック内に1文しかなく、その文が短い場合であれば
   それの中括弧をなくし、1文で書いてもよい。
   （ 例 ： if (hogeFlag == true) hogeFlag = false; )
