/* フレームワークの説明 */


このフレームワークはOpenGLとSDLを利用し作成されたゲーム用のフレームワークです。


サポートされているもの
＝＝＝＝＝＝＝＝＝＝＝＝

・ 2D・3D描画
・ 入力( キーボード・マウス・ゲームパッド( XInput ) )
・ サウンド
・ テクスチャ
・ シェーダー( GLSL )
・ モデル( OBJ・FBX( アニメーション有効 )・自作フォーマット )
・ 物理エンジン( BulletPhysics )、
・ スクリプト( Lua )
・ パーティクルシステム
・ 数学( Vector・Matrix・Quaternion・Easing・Fuzzy … )
・ フォント( TTF )
・ AI( 有限オートマトン・ニューラルネットワーク )
・ タスクシステム
・ シーン管理
・ ゲームオブジェクト管理とメッセージング
・ タイマー
・ メモリ管理( メモリプール )
・ 衝突判定・空間分割( 4分木 )


… 等々


※ 注

・ ウィンドウは複数作成できますが、OpenGLを適応できるのはメインウィンドウのみです。
・ 基本的にスレッドセーフでないです。
・ テクスチャユニットは最大25まで使用できます。( 26 〜 31はフレームワーク側で占有している )
